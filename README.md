# Surf Coffee Landing

Лендинг в стиле Surf Coffee: mobile-first, адаптив от 320px, без внешних библиотек.

## Запуск

Откройте `index.html` в браузере или разверните на Vercel.

## Изображения

Для полного отображения добавьте в папку `images/` файлы:

- `city-skyline.jpg` — фон hero-секции
- `new-drip.jpg`, `team-table.jpg` — карточки «Новинка» и «Команда»
- `mixtape-coffee.jpg`, `drip-box.jpg`, `brownie.jpg` — карточки брендов
- `cafe-interior.jpg` — блок «Открой кофейню»
- `spot-sunday.jpg`, `spot-bove.jpg`, `spot-kazan.jpg` — споты

Без них страница отображается с серыми плейсхолдерами.
